# 用电效率说明

Indevolt 微型储能可以把光伏发电或市电存入电池，并在家庭需要时释放使用。在这个过程中，并非所有电能都能被完整利用，因此就有了“效率”的概念。

## 什么是用电效率？

用电效率表示：**输入的电能中，有多少最终被成功利用。**

一句话理解：
> 储能效率 = 输出电量 ÷ 输入电量

例如：
- 往电池里充入 10 kWh 电能  
- 实际可用电量为 9 kWh  

👉 那么系统效率为 90%（约 10% 能量在过程中损耗）


:::info **为什么会有损耗？**

储能设备不是“无损传输”，电在流动和转换过程中会有损失，主要来自：

- 电池本身损耗：充电、放电过程中会发热，属于化学反应不可避免的损耗
- 逆变转换损耗（DC ↔ AC）：家用设备通常要把直流电变成交流电，每次转换都会损失一部分能量
- 系统损耗**：包括线路、电路元件以及设备自身运行消耗
:::

## 不同使用方式下的效率差异

不同的用电路径，会直接影响整体效率。一般来说：

> 转换环节越多 → 损耗越大 → 效率越低

下面是三种典型使用场景：

### 1. 光伏电先存入电池再使用（PV → BAT → AC）

**使用场景：**
白天光伏发电产生的电，如果家庭用不完，会先存入电池，晚上或光伏不足时再用电池供家庭使用。

**损耗说明**

- 电池充电损耗（受光照波动影响）  
- 电池放电损耗  
- 逆变器转换损耗  

**典型效率示例：** 

光伏 (800W) → 电池 → 交流输出 (800W)


| 型号           | 效率   |
| -------------- | ------ |
| PowerFlex 2000 | ≥78.8% |
| SolidFlex 2000 | ≥78.8% |


### 2. 市电先存入电池再使用（AC → BAT → AC）

**使用场景：**
电网电价低时把市电存入电池，高价时再用，或者当作备用电源。

**损耗说明：**  

- AC → DC 转换损耗  
- 电池充放电损耗  
- DC → AC 转换损耗  

**典型效率示例：**  

市电 (800W) → 电池 → 交流输出(800W)

| 型号           | 效率   |
| -------------- | ------ |
| PowerFlex 2000 | ≥80.2% |
| SolidFlex 2000 | ≥80.2% |


### 3. 光伏电直接供家庭使用（PV → AC）

**使用场景：**
白天家庭用电刚好匹配光伏发电量，电能不经过电池直接使用。

**损耗说明**

- 不经过电池充放电，能量损耗最少  
- 只有逆变器转换和 MPPT 跟踪会有少量损耗  


**典型效率示例：**  

光伏 → 逆变器 → 交流输出（满载）

| 型号           | 效率 |
| -------------- | ---- |
| PowerFlex 2000 | ≥93% |
| SolidFlex 2000 | ≥93% |
