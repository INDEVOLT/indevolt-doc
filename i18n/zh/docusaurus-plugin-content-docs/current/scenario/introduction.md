# 微储应用场景总览

微储适用于各种家庭和小型商业场景，支持灵活配置与自动运行，能够满足多种用电需求。


**核心优势：**
- 按需组合：根据具体需求选择组件
- 自动运行：智能管理，节能高效
- 多种接入方式：支持多样化的设备和配置

## 系统组合方式

根据不同需求，系统可以根据以下维度进行自由组合，灵活适配各种使用场景：


| 维度        | 说明        |
| --------------- | --------------------------------- |
| **并网 / 离网** | ◽并网：系统接入电网运行，正常情况下与电网协同工作；当电网停电时，可自动切换至离网备电模式运行（切换时间 < 20 ms）<br />◽离网：独立运行，适用于没有电网接入的场所                                    |
| **相数**        | ◽单相系统：适用于家庭或小型场所<br />◽三相系统：适用于大功率需求的场所                                                                  |
| **系统结构**    | ◽单机：基础配置<br />◽并机：扩展容量<br /> <br />*注：系统功率与设备配置数量有关，单台主机功率有限，可根据需求扩展电池包或增加主机数量（并机）。* |
| **光伏接入**    | ◽无存量光伏：无光伏设备，适用于基础配置<br />◽阳台光伏：小型光伏设备（微逆通过微储 Backup 口接入）<br />◽户用光伏：适用于家庭屋顶光伏（直连微储 PV 接口）      |
| **功率范围**    | 功率限制因国家不同而有所差异。例如，在德国，插拔式设备的最大功率限制为 800W                                                            |
| **外设扩展**    | ◽智能电表：监测与控制电能使用<br />◽智能插座：负载管理与控制<br />◽备电箱：在断电时提供备用电源，确保系统连续供电   |
